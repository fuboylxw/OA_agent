## 🚀 快速开始

### 方式1：自动化脚本（推荐）
```bash
cd OA_agent
./setup.sh
pnpm dev
```

### 方式2：Docker一键启动
```bash
cd OA_agent
docker compose up --build
```

### 方式3：手动启动
```bash
cd OA_agent
pnpm install
cp .env.example .env
docker compose up -d postgres redis minio
pnpm db:migrate
pnpm db:seed
pnpm build
pnpm dev
```

- **前端**: http://localhost:3000

## 🎓 使用场景

### 场景1：新OA系统接入
```
1. 管理员访问初始化中心
2. 提供OA系统URL或OpenAPI规范
3. 系统自动识别并生成OCL报告
4. 审核通过后发布到流程库
5. 用户即可通过对话发起申请
```

### 场景2：用户发起申请
```
1. 用户: "我要报销差旅费1000元"
2. 系统识别意图并匹配流程
3. 系统提取金额，询问事由
4. 用户: "出差北京"
5. 系统询问日期
6. 用户: "2024-03-15"
7. 系统生成草稿，用户确认
8. 提交到OA系统
```

### 场景3：查询进度
```
1. 用户: "我的请假申请到哪了"
2. 系统识别查询意图
3. 返回申请列表和状态
4. 显示完整时间线
```
