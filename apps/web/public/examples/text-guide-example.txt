# 全局
入口链接: https://oa.example.com/workbench
执行方式: browser

# 共享步骤
点击 登录工作台

## 流程: 请假申请
流程编码: leave_request
参数:
- 开始日期 | date | 必填
- 结束日期 | date | 必填
- 请假类型 | select | 必填
- 请假原因 | textarea | 必填
步骤:
- 点击 申请中心
- 点击 请假申请
- 选择 请假类型
- 输入 开始日期
- 输入 结束日期
- 输入 请假原因
- 点击 提交
- 看到 已提交 就结束
测试样例:
- 请假类型: 年假
- 开始日期: 2026-04-01
- 结束日期: 2026-04-02
- 请假原因: 家中有事

## 流程: 报销申请
流程编码: expense_submit
参数:
- 金额 | number | 必填
- 说明 | textarea | 必填
- 附件 | file | 选填
步骤:
- 点击 申请中心
- 点击 报销申请
- 输入 金额
- 输入 说明
- 上传 附件
- 点击 提交
- 看到 提交成功 就结束
测试样例:
- 金额: 1200
- 说明: 客户拜访交通费
