八、快速开始
================================================================================

方式1: 自动化脚本 (推荐)
  cd OA_agent
  ./setup.sh
  pnpm dev

方式2: Docker一键启动
  cd OA_agent
  docker compose up --build

方式3: 验证脚本
  cd OA_agent
  ./verify.sh

访问地址:
  - 前端: http://localhost:3000
  - API: http://localhost:3001
  - API文档: http://localhost:3001/api/docs
  - 健康检查: http://localhost:3001/health
