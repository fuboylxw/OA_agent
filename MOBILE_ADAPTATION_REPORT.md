## 🔧 技术实现

### 1. 新增状态管理

```typescript
const [sidebarOpen, setSidebarOpen] = useState(false);
```

用于控制移动端抽屉的打开/关闭状态。

### 2. 移动端抽屉组件

#### 遮罩层
```tsx
{sidebarOpen && (
  <div
    className="fixed inset-0 bg-black/40 z-40 lg:hidden transition-opacity"
    onClick={() => setSidebarOpen(false)}
  />
)}
```

- `fixed inset-0`：全屏覆盖
- `bg-black/40`：40% 透明度黑色背景
- `z-40`：层级低于抽屉面板
- `lg:hidden`：桌面端隐藏
- `transition-opacity`：淡入淡出动画

#### 抽屉面板
```tsx
<div
  className={`fixed inset-y-0 left-0 w-72 bg-white z-50 lg:hidden flex flex-col transition-transform duration-300 ${
    sidebarOpen ? 'translate-x-0' : '-translate-x-full'
  }`}
>
```

- `fixed inset-y-0 left-0`：固定在左侧，全高
- `w-72`：宽度 288px（18rem）
- `z-50`：层级高于遮罩层
- `transition-transform duration-300`：300ms 滑动动画
- `translate-x-0` / `-translate-x-full`：滑入/滑出效果

#### 自动关闭逻辑
所有交互操作都添加了 `setSidebarOpen(false)`：

```tsx
// 新建对话
onClick={() => {
  createNewChat();
  setSidebarOpen(false);
}}

// 快捷操作
onClick={() => {
  sendMessage(action.message);
  setSidebarOpen(false);
}}

// 加载会话
onClick={() => {
  loadSession(session.id);
  setSidebarOpen(false);
}}
```

### 3. 汉堡菜单按钮

```tsx
<button
  onClick={() => setSidebarOpen(true)}
  className="w-9 h-9 rounded-lg hover:bg-gray-100 flex items-center justify-center transition-colors lg:hidden"
>
  <i className="fas fa-bars text-gray-600 text-sm"></i>
</button>
```

- 位置：聊天头部左侧，robot 图标之前
- 尺寸：36×36px（满足触控目标要求）
- `lg:hidden`：桌面端自动隐藏

### 4. 响应式网格

```tsx
<div className="grid grid-cols-1 sm:grid-cols-2 gap-2.5 w-full max-w-md">
```

- `< 640px`：单列布局（`grid-cols-1`）
- `≥ 640px`：双列布局（`sm:grid-cols-2`）

### 5. 触控优化

#### 输入框和按钮
```tsx
// 输入框
className="flex-1 px-3.5 py-3 ..."  // py-2.5 → py-3

// 发送按钮
className="... px-5 py-3 ..."  // py-2.5 → py-3
```

- `py-3` = 12px padding = 48px 总高度
- 满足 iOS 人机界面指南的 44px 最小触控目标

#### 安全区域适配
```tsx
<div
  className="border-t border-gray-200 bg-white flex-shrink-0"
  style={{ paddingBottom: 'env(safe-area-inset-bottom, 0px)' }}
>
```

- 使用 CSS 环境变量 `env(safe-area-inset-bottom)`
- 适配 iPhone X+ 的底部 Home Indicator
- 避免输入区域被遮挡
