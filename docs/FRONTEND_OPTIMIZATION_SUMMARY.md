## 一、设计理念

### 1.1 设计方向

采用**现代简约 + 科技感**的设计风格，具有以下特点：

- **深色主题**：深蓝色背景 (#0f172a) 营造专业科技感
- **渐变色彩**：紫色 (#6366f1) + 粉色 (#ec4899) + 橙色 (#f59e0b) 渐变，富有活力
- **毛玻璃效果**：backdrop-filter 实现半透明毛玻璃质感
- **流畅动画**：CSS 动画实现淡入、上浮、悬停等交互效果
- **独特字体**：使用 Noto Sans SC 和 Noto Serif SC，避免通用字体

### 1.2 设计差异化

**避免的通用 AI 美学**：
- ❌ Inter、Roboto、Arial 等通用字体
- ❌ 紫色渐变 + 白色背景的陈词滥调
- ❌ 千篇一律的卡片布局
- ❌ 缺乏个性的设计

**采用的独特设计**：
- ✅ Noto Sans SC 中文字体 + Noto Serif SC 衬线字体
- ✅ 深色主题 + 多色渐变背景
- ✅ 毛玻璃效果 + 渐变边框
- ✅ 丰富的动画和微交互
- ✅ 图标 + Emoji 结合的视觉语言

## 三、技术实现

### 3.1 CSS 技术

1. **CSS 变量**：统一颜色和样式管理
2. **CSS Grid**：响应式布局
3. **Flexbox**：灵活的组件布局
4. **CSS 动画**：@keyframes 定义动画
5. **backdrop-filter**：毛玻璃效果
6. **渐变**：linear-gradient、radial-gradient
7. **伪元素**：::before、::after 实现装饰效果

### 3.2 React 技术

1. **useState**：状态管理
2. **useEffect**：数据加载
3. **useRef**：滚动控制
4. **条件渲染**：空状态、加载状态
5. **事件处理**：点击、输入、键盘事件

### 3.3 响应式设计

- **移动端**：单列布局
- **平板**：2 列布局
- **桌面**：3-4 列布局
- **导航栏**：移动端隐藏部分链接

```css

```

- **Noto Sans SC**：正文字体（无衬线）
- **Noto Serif SC**：标题字体（衬线）
