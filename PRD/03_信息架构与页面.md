## 3.4 页面状态设计

### 通用状态

| 状态 | 表现 | 触发条件 |
|------|------|----------|
| 加载中 | 3个脉冲点 + "加载中..." | 数据请求中 |
| 空状态 | 图标 + 提示文字 + CTA按钮 | 列表为空 |
| 错误状态 | 错误图标 + 错误信息 + 重试按钮 | 请求失败 |
| 成功状态 | 绿色提示条 | 操作成功 |

### 对话页特殊状态

| 状态 | 表现 |
|------|------|
| 空对话 | 欢迎语 + 4个快捷操作卡片 |
| 等待回复 | 3个脉冲点 + "正在思考..." |
| 草稿确认 | 表单预览卡片 + 确认/修改按钮 |
| 提交成功 | 成功图标 + 申请编号 + 查看详情链接 |
