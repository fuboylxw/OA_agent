# OA智能体上下文管理设计

## 设计目标

参考 OpenAI Agent 设计模式，实现：
1. **上下文隔离**：每个会话独立，避免信息泄露和混淆
2. **上下文共享**：跨会话共享用户信息、偏好设置、历史记录
3. **高效访问**：快速读取和更新上下文数据
4. **可扩展性**：支持未来新增上下文类型

## 上下文分层架构

```
┌─────────────────────────────────────────────────────┐
│                   用户请求                           │
└─────────────────────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────────┐
│              ContextManager (上下文管理器)           │
│  ┌─────────────────────┬─────────────────────────┐  │
│  │   Isolated Context  │   Shared Context        │  │
│  │   (会话级隔离)       │   (用户级共享)           │  │
│  └─────────────────────┴─────────────────────────┘  │
└─────────────────────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────────┐
│                 Agent 执行层                         │
│  IntentAgent → FlowAgent → FormAgent → MCP Tools   │
└─────────────────────────────────────────────────────┘
```

## 上下文类型定义

### 1. 隔离上下文 (Isolated Context)

**生命周期**：单次会话（Session）

**存储位置**：`ChatSession.metadata`

**数据结构**：
```typescript
interface IsolatedContext {
  // 会话标识
  sessionId: string;

  // 对话历史（仅当前会话）
  conversationHistory: Array<{
    role: 'user' | 'assistant';
    content: string;
    timestamp: Date;
  }>;

  // 当前流程状态
  currentProcessCode?: string;
  currentProcessName?: string;

  // 当前表单数据
  currentFormData?: Record<string, any>;

  // 待提交草稿
  pendingDraftId?: string;

  // 澄清状态
  clarificationState?: {
    type: 'flow_selection' | 'field_input';
    candidates?: Array<{
      code: string;
      name: string;
      confidence: number;
    }>;
    missingFields?: Array<{
      key: string;
      label: string;
    }>;
  };

  // 会话阶段
  stage: 'intent_recognition' | 'flow_matching' | 'form_filling' | 'confirmation' | 'completed';

  // 临时标记
  flags?: {
    needsConfirmation?: boolean;
    isRetrying?: boolean;
    errorCount?: number;
  };
}
```

**特点**：
- 会话结束后清空
- 不跨会话共享
- 用于追踪单次申请的完整流程

### 2. 共享上下文 (Shared Context)

**生命周期**：用户级持久化

**存储位置**：独立的 `UserContext` 表或 Redis

**数据结构**：
```typescript
interface SharedContext {
  // 用户标识
  userId: string;
  tenantId: string;

  // 用户基本信息
  userInfo: {
    name: string;
    employeeId: string;
    department: string;
    position: string;
    email: string;
    phone?: string;
  };

  // 常用流程模板
  favoriteProcesses: Array<{
    processCode: string;
    processName: string;
    usageCount: number;
    lastUsedAt: Date;
  }>;

  // 表单预填充模板
  formTemplates: Record<string, Record<string, any>>;
  // 例如：
  // {
  //   'travel_expense': {
  //     'department': '技术部',
  //     'costCenter': 'CC001'
  //   }
  // }

  // 用户偏好设置
  preferences: {
    language: 'zh-CN' | 'en-US';
    confirmBeforeSubmit: boolean;
    autoFillFromHistory: boolean;
    notificationEnabled: boolean;
  };

  // 历史申请摘要（最近N条）
  recentSubmissions: Array<{
    submissionId: string;
    processCode: string;
    processName: string;
    status: string;
    submittedAt: Date;
    formDataSummary?: string; // 简要描述
  }>;

  // 学习到的用户习惯
  learnedPatterns?: {
    commonReasons?: Record<string, string[]>; // 流程 -> 常用原因
    preferredOptions?: Record<string, any>; // 字段 -> 偏好选项
  };

  // 上下文版本（用于缓存失效）
  version: number;
  updatedAt: Date;
}
```

**特点**：
- 跨会话持久化
- 用于提升用户体验（自动填充、智能推荐）
- 需要定期更新和维护

### 3. 全局上下文 (Global Context)

**生命周期**：租户级

**存储位置**：缓存 + 数据库

**数据结构**：
```typescript
interface GlobalContext {
  tenantId: string;

  // 可用流程列表（缓存）
  availableProcesses: Array<{
    processCode: string;
    processName: string;
    category: string;
    enabled: boolean;
  }>;

  // 流程关键词映射（用于匹配优化）
  processKeywords: Record<string, string[]>;

  // 租户配置
  tenantConfig: {
    enableLLM: boolean;
    llmProvider: string;
    maxSessionDuration: number;
    autoSubmitEnabled: boolean;
  };

  // 缓存时间戳
  cachedAt: Date;
}
```

## 上下文管理器实现

### ContextManager 类

**位置**：`apps/api/src/modules/assistant/context/context.manager.ts`

```typescript
export class ContextManager {
  constructor(
    private prisma: PrismaService,
    private redis: RedisService,
  ) {}

  // ========== 隔离上下文管理 ==========

  /**
   * 获取会话的隔离上下文
   */
  async getIsolatedContext(sessionId: string): Promise<IsolatedContext> {
    const session = await this.prisma.chatSession.findUnique({
      where: { id: sessionId },
      select: { metadata: true }
    });

    return session?.metadata as IsolatedContext || this.createEmptyIsolatedContext(sessionId);
  }

  /**
   * 更新隔离上下文
   */
  async updateIsolatedContext(
    sessionId: string,
    updates: Partial<IsolatedContext>
  ): Promise<void> {
    const current = await this.getIsolatedContext(sessionId);
    const merged = { ...current, ...updates };

    await this.prisma.chatSession.update({
      where: { id: sessionId },
      data: { metadata: merged }
    });
  }

  /**
   * 清空隔离上下文（提交完成后）
   */
  async clearIsolatedContext(sessionId: string): Promise<void> {
    await this.prisma.chatSession.update({
      where: { id: sessionId },
      data: { metadata: {} }
    });
  }

  // ========== 共享上下文管理 ==========

  /**
   * 获取用户的共享上下文
   */
  async getSharedContext(userId: string, tenantId: string): Promise<SharedContext> {
    const cacheKey = `shared_context:${tenantId}:${userId}`;

    // 先从 Redis 读取
    const cached = await this.redis.get(cacheKey);
    if (cached) {
      return JSON.parse(cached);
    }

    // 从数据库构建
    const context = await this.buildSharedContext(userId, tenantId);

    // 写入缓存（1小时）
    await this.redis.setex(cacheKey, 3600, JSON.stringify(context));

    return context;
  }

  /**
   * 更新共享上下文
   */
  async updateSharedContext(
    userId: string,
    tenantId: string,
    updates: Partial<SharedContext>
  ): Promise<void> {
    const current = await this.getSharedContext(userId, tenantId);
    const merged = {
      ...current,
      ...updates,
      version: current.version + 1,
      updatedAt: new Date()
    };

    // 持久化到数据库
    await this.prisma.userContext.upsert({
      where: { userId_tenantId: { userId, tenantId } },
      create: { userId, tenantId, data: merged },
      update: { data: merged }
    });

    // 更新缓存
    const cacheKey = `shared_context:${tenantId}:${userId}`;
    await this.redis.setex(cacheKey, 3600, JSON.stringify(merged));
  }

  /**
   * 记录用户行为（用于学习）
   */
  async recordUserBehavior(
    userId: string,
    tenantId: string,
    behavior: {
      processCode: string;
      formData: Record<string, any>;
      submittedAt: Date;
    }
  ): Promise<void> {
    const context = await this.getSharedContext(userId, tenantId);

    // 更新常用流程
    const favorite = context.favoriteProcesses.find(
      f => f.processCode === behavior.processCode
    );
    if (favorite) {
      favorite.usageCount++;
      favorite.lastUsedAt = behavior.submittedAt;
    } else {
      context.favoriteProcesses.push({
        processCode: behavior.processCode,
        processName: '', // 需要查询
        usageCount: 1,
        lastUsedAt: behavior.submittedAt
      });
    }

    // 学习常用原因
    if (behavior.formData.reason) {
      if (!context.learnedPatterns) {
        context.learnedPatterns = { commonReasons: {} };
      }
      if (!context.learnedPatterns.commonReasons[behavior.processCode]) {
        context.learnedPatterns.commonReasons[behavior.processCode] = [];
      }
      const reasons = context.learnedPatterns.commonReasons[behavior.processCode];
      if (!reasons.includes(behavior.formData.reason)) {
        reasons.push(behavior.formData.reason);
      }
    }

    await this.updateSharedContext(userId, tenantId, context);
  }

  // ========== 全局上下文管理 ==========

  /**
   * 获取租户的全局上下文
   */
  async getGlobalContext(tenantId: string): Promise<GlobalContext> {
    const cacheKey = `global_context:${tenantId}`;

    const cached = await this.redis.get(cacheKey);
    if (cached) {
      return JSON.parse(cached);
    }

    const context = await this.buildGlobalContext(tenantId);

    // 缓存 30 分钟
    await this.redis.setex(cacheKey, 1800, JSON.stringify(context));

    return context;
  }

  // ========== 上下文合并 ==========

  /**
   * 为 Agent 合并所有相关上下文
   */
  async mergeContextForAgent(
    sessionId: string,
    userId: string,
    tenantId: string
  ): Promise<{
    isolated: IsolatedContext;
    shared: SharedContext;
    global: GlobalContext;
  }> {
    const [isolated, shared, global] = await Promise.all([
      this.getIsolatedContext(sessionId),
      this.getSharedContext(userId, tenantId),
      this.getGlobalContext(tenantId)
    ]);

    return { isolated, shared, global };
  }

  // ========== 私有辅助方法 ==========

  private createEmptyIsolatedContext(sessionId: string): IsolatedContext {
    return {
      sessionId,
      conversationHistory: [],
      stage: 'intent_recognition',
      flags: {}
    };
  }

  private async buildSharedContext(
    userId: string,
    tenantId: string
  ): Promise<SharedContext> {
    // 从数据库查询用户信息
    const user = await this.prisma.user.findUnique({
      where: { id: userId },
      include: { profile: true }
    });

    // 查询最近提交记录
    const recentSubmissions = await this.prisma.submission.findMany({
      where: { userId, tenantId },
      orderBy: { submittedAt: 'desc' },
      take: 10,
      include: { template: true }
    });

    // 查询或创建用户上下文
    const userContext = await this.prisma.userContext.findUnique({
      where: { userId_tenantId: { userId, tenantId } }
    });

    return {
      userId,
      tenantId,
      userInfo: {
        name: user.name,
        employeeId: user.profile?.employeeId || '',
        department: user.profile?.department || '',
        position: user.profile?.position || '',
        email: user.email,
        phone: user.profile?.phone
      },
      favoriteProcesses: userContext?.data?.favoriteProcesses || [],
      formTemplates: userContext?.data?.formTemplates || {},
      preferences: userContext?.data?.preferences || {
        language: 'zh-CN',
        confirmBeforeSubmit: true,
        autoFillFromHistory: true,
        notificationEnabled: true
      },
      recentSubmissions: recentSubmissions.map(s => ({
        submissionId: s.id,
        processCode: s.template.processCode,
        processName: s.template.processName,
        status: s.status,
        submittedAt: s.submittedAt,
        formDataSummary: this.summarizeFormData(s.formData)
      })),
      learnedPatterns: userContext?.data?.learnedPatterns,
      version: userContext?.data?.version || 1,
      updatedAt: userContext?.updatedAt || new Date()
    };
  }

  private async buildGlobalContext(tenantId: string): Promise<GlobalContext> {
    const processes = await this.prisma.processLibrary.findMany({
      where: { tenantId, enabled: true },
      select: {
        processCode: true,
        processName: true,
        category: true,
        enabled: true
      }
    });

    const config = await this.prisma.tenantConfig.findUnique({
      where: { tenantId }
    });

    return {
      tenantId,
      availableProcesses: processes,
      processKeywords: this.buildProcessKeywords(processes),
      tenantConfig: {
        enableLLM: config?.enableLLM ?? true,
        llmProvider: config?.llmProvider || 'openai',
        maxSessionDuration: config?.maxSessionDuration || 3600,
        autoSubmitEnabled: config?.autoSubmitEnabled ?? false
      },
      cachedAt: new Date()
    };
  }

  private buildProcessKeywords(
    processes: Array<{ processCode: string; processName: string }>
  ): Record<string, string[]> {
    // 预定义关键词映射
    const predefined = {
      travel_expense: ['差旅', '报销', '出差', '机票', '酒店'],
      leave_request: ['请假', '休假', '年假', '病假', '事假'],
      // ... 更多
    };

    // 合并流程名称分词
    const result = { ...predefined };
    processes.forEach(p => {
      if (!result[p.processCode]) {
        result[p.processCode] = [p.processName];
      }
    });

    return result;
  }

  private summarizeFormData(formData: any): string {
    // 提取关键字段生成摘要
    const keys = ['reason', 'amount', 'startDate', 'endDate'];
    const summary = keys
      .filter(k => formData[k])
      .map(k => `${k}: ${formData[k]}`)
      .join(', ');
    return summary || '无摘要';
  }
}
```

## Agent 中的上下文使用

### IntentAgent 使用示例

```typescript
export class IntentAgent {
  async recognizeIntent(
    message: string,
    context: {
      isolated: IsolatedContext;
      shared: SharedContext;
      global: GlobalContext;
    }
  ): Promise<IntentResult> {
    // 利用共享上下文优化意图识别
    const recentProcesses = context.shared.favoriteProcesses
      .slice(0, 3)
      .map(f => f.processCode);

    // 利用隔离上下文理解对话历史
    const conversationContext = context.isolated.conversationHistory
      .slice(-3)
      .map(h => `${h.role}: ${h.content}`)
      .join('\n');

    const prompt = `
用户最近常用流程: ${recentProcesses.join(', ')}
对话历史:
${conversationContext}

当前用户消息: ${message}

请识别用户意图...
`;

    // 调用 LLM
    const result = await this.llmClient.complete(prompt);

    return result;
  }
}
```

### FormAgent 使用示例

```typescript
export class FormAgent {
  async extractFields(
    processCode: string,
    schema: FormSchema,
    message: string,
    context: {
      isolated: IsolatedContext;
      shared: SharedContext;
    }
  ): Promise<FormResult> {
    // 从共享上下文获取预填充模板
    const template = context.shared.formTemplates[processCode] || {};

    // 从隔离上下文获取已填充字段
    const currentData = context.isolated.currentFormData || {};

    // 合并优先级: 当前输入 > 已填充 > 模板 > 默认值
    const extractedFields = await this.extractFromMessage(message, schema);

    const mergedData = {
      ...template,        // 最低优先级
      ...currentData,     // 中等优先级
      ...extractedFields  // 最高优先级
    };

    return {
      extractedFields: mergedData,
      missingFields: this.findMissingFields(schema, mergedData),
      isComplete: this.checkComplete(schema, mergedData)
    };
  }
}
```

async function cleanupExpiredSessions() {
  const expiredSessions = await prisma.chatSession.findMany({
    where: {
      updatedAt: { lt: new Date(Date.now() - 24 * 60 * 60 * 1000) },
      metadata: { not: {} }
    }
  });

  for (const session of expiredSessions) {
    await contextManager.clearIsolatedContext(session.id);
  }
}

## 性能优化策略

### 1. 缓存策略

- **共享上下文**: Redis 缓存 1 小时，写时更新
- **全局上下文**: Redis 缓存 30 分钟，定时刷新
- **隔离上下文**: 直接读写数据库（频繁变更）

### 2. 懒加载

```typescript
// 只在需要时加载共享上下文
if (needsAutoFill) {
  const sharedContext = await contextManager.getSharedContext(userId, tenantId);
  // 使用 formTemplates
}
```

### 3. 批量操作

```typescript
// 一次性获取所有上下文
const allContext = await contextManager.mergeContextForAgent(
  sessionId, userId, tenantId
);
```

## 安全与隐私

### 1. 数据隔离

- 租户级隔离：所有查询必须带 `tenantId`
- 用户级隔离：共享上下文仅限本人访问

### 2. 敏感信息处理

```typescript
// 不在上下文中存储敏感信息
const sanitizedFormData = {
  ...formData,
  password: undefined,
  idCard: maskIdCard(formData.idCard)
};
```

### 3. 上下文清理

- 会话结束后立即清空隔离上下文
- 定期清理过期的共享上下文数据
