## 9.1 核心业务指标（North Star Metrics）

| 指标 | 定义 | 目标值 | 统计周期 |
|------|------|--------|----------|
| MAU | 月活跃用户数 | Month 6: 1,000 / Month 12: 50,000 | 月 |
| 对话成功率 | 对话产出草稿的比例 | ≥ 85% | 日 |
| 提交成功率 | 提交到OA成功的比例 | ≥ 90% | 日 |
| 首次提交成功率 | 无需退回修改的比例 | ≥ 85% | 周 |
| 平均办事耗时 | 从对话开始到提交完成 | ≤ 6分钟 | 日 |

## 9.2 产品指标

### 用户增长

| 指标 | 定义 | 计算方式 |
|------|------|----------|
| 新增用户数 | 首次登录的用户数 | COUNT(DISTINCT user_id) WHERE first_login = today |
| DAU | 日活跃用户数 | COUNT(DISTINCT user_id) WHERE login_date = today |
| WAU | 周活跃用户数 | COUNT(DISTINCT user_id) WHERE login_date IN last_7_days |
| MAU | 月活跃用户数 | COUNT(DISTINCT user_id) WHERE login_date IN last_30_days |
| DAU/MAU | 用户粘性 | DAU / MAU |

### 用户留存

| 指标 | 定义 | 目标 |
|------|------|------|
| 次日留存 | 注册次日回访比例 | ≥ 50% |
| 7日留存 | 注册7日内回访比例 | ≥ 30% |
| 30日留存 | 注册30日内回访比例 | ≥ 20% |
| 周留存 | 本周活跃用户下周回访比例 | ≥ 60% |
| 月留存 | 本月活跃用户下月回访比例 | ≥ 40% |

### 功能使用

| 指标 | 定义 | 目标 |
|------|------|------|
| 对话发起次数 | 每日对话总数 | 持续增长 |
| 人均对话次数 | 每用户每日平均对话数 | 1-3次 |
| 意图识别准确率 | 正确识别意图的比例 | ≥ 85% |
| 字段自动填充率 | 自动填充字段占总字段比例 | ≥ 60% |
| 平均对话轮次 | 完成一次办事的平均对话轮数 | ≤ 4轮 |

## 9.3 技术指标

### 性能指标

| 指标 | 定义 | 目标 | 告警阈值 |
|------|------|------|----------|
| API P50 | 50分位响应时间 | ≤ 500ms | > 1s |
| API P95 | 95分位响应时间 | ≤ 3s | > 5s |
| API P99 | 99分位响应时间 | ≤ 5s | > 10s |
| 错误率 | 5xx错误占比 | ≤ 0.1% | > 1% |
| 可用性 | 成功请求占比 | ≥ 99.9% | < 99.5% |

### 基础设施指标

| 指标 | 定义 | 告警阈值 |
|------|------|----------|
| CPU使用率 | 服务器CPU占用 | > 80% |
| 内存使用率 | 服务器内存占用 | > 85% |
| 磁盘使用率 | 磁盘空间占用 | > 90% |
| DB连接数 | 数据库活跃连接 | > 80% of pool |
| Redis内存 | Redis内存占用 | > 80% |
| 队列积压 | BullMQ待处理任务数 | > 1000 |

### 业务健康指标

| 指标 | 定义 | 告警阈值 |
|------|------|----------|
| Bootstrap成功率 | 初始化任务成功比例 | < 80% |
| OA连接器健康率 | 健康检查通过比例 | < 90% |
| 提交失败率 | 提交到OA失败比例 | > 5% |
| 权限拒绝率 | 权限校验拒绝比例 | > 30%（异常高） |
| 审计日志写入延迟 | 日志写入延迟 | > 5s |

## 9.4 埋点方案

### 前端埋点

#### 页面访问（PV/UV）

| 事件名 | 触发时机 | 参数 |
|--------|----------|------|
| page_view | 页面加载完成 | page_path, user_id, tenant_id |
| page_leave | 页面离开 | page_path, duration_ms |

#### 用户行为

| 事件名 | 触发时机 | 参数 |
|--------|----------|------|
| chat_send | 发送对话消息 | session_id, message_length |
| chat_quick_action | 点击快捷操作 | action_type |
| draft_confirm | 确认草稿提交 | process_code, field_count |
| draft_modify | 修改草稿 | process_code, modified_fields |
| submission_view | 查看申请详情 | submission_id |
| submission_urge | 点击催办 | submission_id |
| submission_cancel | 点击撤回 | submission_id |
| process_browse | 浏览流程库 | category |
| process_search | 搜索流程 | search_term, result_count |
| bootstrap_create | 创建初始化任务 | source_type |
| connector_health | 执行健康检查 | connector_id |
| login_submit | 提交登录 | - |
| login_success | 登录成功 | user_id |
| login_fail | 登录失败 | error_type |

#### 性能埋点

| 事件名 | 触发时机 | 参数 |
|--------|----------|------|
| api_request | API请求发出 | endpoint, method |
| api_response | API响应收到 | endpoint, status, duration_ms |
| api_error | API请求失败 | endpoint, error_code, error_message |
| page_load | 页面加载完成 | page_path, load_time_ms |
| first_paint | 首次绘制 | fp_ms |
| first_contentful_paint | 首次内容绘制 | fcp_ms |

### 后端埋点

## 9.6 告警规则

| 告警名称 | 条件 | 级别 | 通知方式 |
|----------|------|------|----------|
| API高错误率 | 5xx > 1% 持续5分钟 | P0 | 短信+邮件 |
| API高延迟 | P95 > 5s 持续5分钟 | P1 | 邮件 |
| 服务不可用 | 健康检查失败 | P0 | 短信+邮件+电话 |
| 数据库连接耗尽 | 连接数 > 90% | P1 | 邮件 |
| 队列积压 | 待处理 > 1000 | P1 | 邮件 |
| 提交失败率高 | 失败率 > 5% 持续10分钟 | P1 | 邮件 |
| OA连接器异常 | 健康检查失败 | P2 | 邮件 |
| 磁盘空间不足 | 使用率 > 90% | P1 | 邮件 |
