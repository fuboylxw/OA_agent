## 6.1 ER关系总览

```
tenant ─1:N─ user
tenant ─1:N─ oa_connector ─1:N─ process_template ─1:N─ process_field
tenant ─1:N─ bootstrap_job ─1:N─ bootstrap_source
                           ─1:N─ bootstrap_report
                           ─1:N─ flow_ir / field_ir / rule_ir / permission_ir
                           ─1:N─ adapter_build ─1:N─ replay_case ─1:N─ replay_result
tenant ─1:N─ chat_session ─1:N─ chat_message
tenant ─1:N─ submission ─1:N─ submission_event
tenant ─1:N─ audit_log

process_template ─1:N─ process_step ─1:N─ ui_action_graph
user ─N:M─ role（通过user.roles JSON数组）
```

## 6.2 核心表定义

### 6.2.1 租户与用户

#### tenant（租户）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK, 自动生成 | 租户ID |
| code | VARCHAR(50) | UNIQUE, NOT NULL | 租户编码 |
| name | VARCHAR(200) | NOT NULL | 租户名称 |
| status | VARCHAR(20) | NOT NULL, DEFAULT 'active' | 状态：active/inactive |
| config | JSONB | | 租户配置 |
| created_at | TIMESTAMP | NOT NULL, DEFAULT NOW() | 创建时间 |
| updated_at | TIMESTAMP | NOT NULL, 自动更新 | 更新时间 |

#### user（用户）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | UUID | PK, 自动生成 | 用户ID |
| tenant_id | UUID | FK→tenant.id, NOT NULL | 租户ID |
| username | VARCHAR(100) | NOT NULL | 用户名 |
| email | VARCHAR(200) | | 邮箱 |
| display_name | VARCHAR(200) | | 显示名称 |
| roles | JSONB | NOT NULL, DEFAULT '[]' | 角色列表 |
| status | VARCHAR(20) | NOT NULL, DEFAULT 'active' | 状态 |
| created_at | TIMESTAMP | NOT NULL | 创建时间 |
| updated_at | TIMESTAMP | NOT NULL | 更新时间 |

**唯一索引**: `(tenant_id, username)`

## 6.3 数据库约束汇总

### 全局约束

1. 所有业务表必须包含 `tenant_id`
2. 所有表必须包含 `created_at` 和 `updated_at`
3. 所有主键使用 UUID
4. 所有时间戳使用 UTC
5. `audit_log` 表禁止 UPDATE 和 DELETE（append-only）

### 唯一约束

| 表 | 唯一索引 |
|------|----------|
| tenant | (code) |
| user | (tenant_id, username) |
| process_template | (process_code, version, tenant_id) |
| submission | (idempotency_key, tenant_id) |

### 外键约束

| 子表 | 外键字段 | 父表 | 级联策略 |
|------|----------|------|----------|
| user | tenant_id | tenant | RESTRICT |
| oa_connector | tenant_id | tenant | RESTRICT |
| process_template | connector_id | oa_connector | SET NULL |
| process_field | template_id | process_template | CASCADE |
| process_rule | template_id | process_template | CASCADE |
| bootstrap_job | tenant_id | tenant | RESTRICT |
| bootstrap_report | job_id | bootstrap_job | CASCADE |
| submission | template_id | process_template | RESTRICT |
| submission_event | submission_id | submission | CASCADE |
| chat_session | user_id | user | RESTRICT |
| chat_message | session_id | chat_session | CASCADE |
